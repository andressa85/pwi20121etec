Estrutura do HTML e Tags

O HTML possui uma estrutura que é composta por tags. As tags possuem funções diferentes entre si, cada uma faz algo específico dentro do código. A seguir está a estrutura básica do HTML:

    Tipos de Tags

  • Tag < h >

    • Esa tag define o tamanho do seu título da sua página, tendo variação em h1,h2,h3,h4,h5,h6. Sendo que h1 é o maior e h6 o menor.


  • Tag < p >

    • Cria parágrafos.
