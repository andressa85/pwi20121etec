    Linguagens Utilizadas

  • HTML

    • É uma linguagem de marcação de hipertexto para web, sendo diferente de uma linguagem de programação, pois não permite criar algo mais dinâmico além de estruturar páginas e documentos para web.
    Os arquivos HTML podem ser lidos por qualquer navegador, seja ele em computador ou celular.

  • CSS

    • Serve para dar estilo nos documentos web. Ele basicamente complementa o HTML em um arquivo externo sendo integrado dentro do HTML, separando a parte visual do documento.
    No CSS, você pode alterar a cor do texto, fundo, fonte etc. Ou seja, ao invés de alterar, por exemplo o fundo dentro do HTML, você pode fazer isso em um arquivo separado, simplificando mais o documento.
